## 1. TOPSIS方法（优劣解距离法）

TOPSIS（Technique for Order Preference by Similarity to Ideal Solution）是一种多属性决策方法。

### 基本步骤

**Step 1: 数据标准化**
- 消除量纲影响，将原始数据转换为无量纲数据
- 常用方法：向量归一化、极差变换等

**Step 2: KMO检验**
- 检验数据是否适合进行因子分析
- KMO值越接近1，表示变量间相关性越强，越适合因子分析
- 通常要求KMO > 0.6

**Step 3: 权重计算**
- 根据标准化后的数据计算各指标权重
- 公式：$w_i = \frac{v_i}{\sqrt{\sum_{j=1}^n v_j^2}}$
- 其中 $v_i$ 为第i个指标的变异系数

### 数学模型

设决策矩阵为 $X = (x_{ij})_{m \times n}$，其中 m 为方案数，n 为指标数。

1. **标准化决策矩阵**：$r_{ij} = \frac{x_{ij}}{\sqrt{\sum_{k=1}^m x_{kj}^2}}$
2. **加权标准化矩阵**：$v_{ij} = w_j \cdot r_{ij}$
3. **确定理想解和负理想解**：
   - 理想解 $A^+ = (v_1^+, v_2^+, ..., v_n^+)$
   - 负理想解 $A^- = (v_1^-, v_2^-, ..., v_n^-)$
4. **计算距离**：
   - 到理想解距离：$D_i^+ = \sqrt{\sum_{j=1}^n (v_{ij} - v_j^+)^2}$
   - 到负理想解距离：$D_i^- = \sqrt{\sum_{j=1}^n (v_{ij} - v_j^-)^2}$
5. **计算相对接近度**：$C_i = \frac{D_i^-}{D_i^+ + D_i^-}$

## 2. 统计检验方法

### 2.1 正态性检验

**Step 1: 样本量判断**
- 样本量3-50：使用Shapiro-Wilk检验
- 样本量>50：使用Jarque-Bera检验
- 辅助方法：QQ图直观判断

**Step 2: 检验方法选择**
- **Shapiro-Wilk检验**：适用于小样本（n ≤ 50）
- **Jarque-Bera检验**：适用于大样本，基于偏度和峰度
- **QQ图**：图形化检验，观察点是否近似直线

**Step 3: 显著性水平**
- 通常取 $\alpha = 0.05$
- 若p值 < 0.05，拒绝原假设（数据不服从正态分布）

### 2.2 相关性检验

#### Pearson相关系数
适用于连续变量且满足正态分布的情况。

**计算公式**：
$r = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^n (x_i - \bar{x})^2 \sum_{i=1}^n (y_i - \bar{y})^2}}$

**假设检验**：
- $H_0: \rho = 0$（无相关性）
- 检验统计量：$t = \frac{r\sqrt{n-2}}{\sqrt{1-r^2}} \sim t(n-2)$

#### Spearman等级相关系数
适用于顺序变量或不满足正态分布的连续变量。

**计算步骤**：
1. 将原始数据转换为等级数据
2. 计算等级差 $d_i = R_{x_i} - R_{y_i}$
3. 计算公式：$\rho = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$

**特点**：
- 对异常值不敏感
- 适用于单调关系

#### Eta系数
用于度量分类变量与连续变量之间的相关性。

## 3. 异常值检测

### IQR方法（四分位距法）

**计算步骤**：

1. **计算四分位数**：
   - $Q_1$：第一四分位数（25%位置）
   - $Q_3$：第三四分位数（75%位置）

2. **计算IQR**：
   $IQR = Q_3 - Q_1$

3. **确定异常值界限**：
   - 下界：$Q_1 - 1.5 \times IQR$
   - 上界：$Q_3 + 1.5 \times IQR$

4. **判断异常值**：
   - 小于下界或大于上界的值为异常值

### 标准化方法（Z-score法）

对于服从正态分布的数据：
- 计算Z-score：$z = \frac{x - \mu}{\sigma}$
- 通常认为 $|z| > 3$ 为异常值

## 4. 假设检验

### t检验

**适用条件**：
1. 样本来自正态总体
2. 样本量较小（n < 30）
3. 总体方差未知

**单样本t检验**：
- 检验样本均值与已知总体均值的差异
- 检验统计量：$t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}$

**两独立样本t检验**：
- 检验两个独立样本的均值差异
- 需要检验方差齐性

**配对样本t检验**：
- 检验同一组对象前后测量的差异

## 6. MATLAB实现示例

### 6.1 TOPSIS方法实现

```matlab
function [score, rank] = topsis(decisionMatrix, weights, benefit)
    % decisionMatrix: 决策矩阵 (m×n)
    % weights: 权重向量 (1×n)
    % benefit: 效益型指标标记 (1×n, 1表示效益型，0表示成本型)
    
    [m, n] = size(decisionMatrix);
    
    % 1. 数据标准化（向量归一化）
    normalized = zeros(m, n);
    for j = 1:n
        normalized(:, j) = decisionMatrix(:, j) / norm(decisionMatrix(:, j));
    end
    
    % 2. 加权标准化
    weighted = normalized .* weights;
    
    % 3. 确定理想解和负理想解
    ideal_best = max(weighted);
    ideal_worst = min(weighted);
    
    % 调整成本型指标
    for j = 1:n
        if benefit(j) == 0  % 成本型指标
            temp = ideal_best(j);
            ideal_best(j) = ideal_worst(j);
            ideal_worst(j) = temp;
        end
    end
    
    % 4. 计算距离
    dist_best = zeros(m, 1);
    dist_worst = zeros(m, 1);
    
    for i = 1:m
        dist_best(i) = sqrt(sum((weighted(i, :) - ideal_best).^2));
        dist_worst(i) = sqrt(sum((weighted(i, :) - ideal_worst).^2));
    end
    
    % 5. 计算相对接近度
    score = dist_worst ./ (dist_best + dist_worst);
    
    % 6. 排序
    [~, rank] = sort(score, 'descend');
end
```

### 6.3 异常值检测（IQR方法）

```matlab
function outliers = detect_outliers_iqr(data)
    % 使用IQR方法检测异常值
    
    Q1 = quantile(data, 0.25);
    Q3 = quantile(data, 0.75);
    IQR = Q3 - Q1;
    
    lower_bound = Q1 - 1.5 * IQR;
    upper_bound = Q3 + 1.5 * IQR;
    
    outliers = data(data < lower_bound | data > upper_bound);
end
```

## 7. 注意事项

1. **方法选择**：
   - 根据数据类型和分布选择合适的方法
   - 正态性检验是许多参数检验的前提

2. **样本量要求**：
   - 小样本使用非参数方法更稳健
   - 大样本时中心极限定理适用

3. **多重比较校正**：
   - 进行多次检验时需要校正显著性水平
   - 常用方法：Bonferroni校正、FDR校正

4. **结果解释**：
   - 统计显著性不等于实际显著性
   - 结合效应量和置信区间综合判断
