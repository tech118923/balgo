# 数学建模常用算法

## 1. 预测类

- **插值与拟合**：小样本内部预测
- **回归分析**：中、大样本内部预测
- **灰色预测**：小样本固定趋势未来预测
- **时间序列**：中、大样本随机因素或周期性特征未来趋势预测
- **神经网络**：大、特大样本未来预测

---

## 2. 评价类（阅读资料构建评价体系）

- **层次分析法**：主观性较强
- **TOPSIS**（理想解法）：假定一个最优解，将已有方案分别与最优解比较
- **聚类分析**：将样本点根据相似性归为一类
- **模糊综合评价**：适用于指标不明确的评价（如大小、胖瘦）
- **灰色关联分析**：计算与理想解间的灰色关联度确定方案的优劣
- **熵权法**：用于客观为评价指标赋权
- **主成分分析法**：面对多个评价指标时找到主要影响因素进行降维（得到主成分后进行解释，合理即可）

### 主成分分析法步骤

**Step 1：指标标准化**
- 分析 → 描述统计 → 描述 → 将标准化结果另存为变量

**Step 2：因子分析**
- 分析 → 降维 → 因子 → 描述（初始解、KMO、球形检验）
- 提取（主成分、碎石图）
- 旋转（无、载荷图）
- 得分（保存为、回归、显示因子得分）

**Step 3：将因子系数转换为主成分系数**
- 在成分矩阵的重新标度成分中：编辑内容 → 在查看器中 → 将数值复制另存为新的变量 → 转换
- 公式：$w_1 = \frac{v_1}{\sqrt{\text{特征值}}}$（特征值在总方差解释中查找）

## 6. 相关性检验

### 变量类型说明

| 类型 | 示例 | 特点 |
| :--- | :--- | :--- |
| **定类** | 男女、民族、婚姻状况 | 仅分类，无顺序 |
| **定序** | 文化程度、工厂规模、年龄大小 | 有顺序，但不能定量 |
| **定距** | 摄氏温度、比率、智力水平 | 无绝对零点 |
| **定比** | 收入、价格 | 有绝对零点 |

### 相关性检验步骤

**Step 1：正态分布检验**

| 样本规模 | 检验方法 |
| :--- | :--- |
| 小样本（3-50） | Shapiro-Wilk 检验 |
| 大样本 | JB 检验 |
| 超大样本 | QQ 图 |

**Step 2：选择相关系数**

| 条件 | 选用方法 |
| :--- | :--- |
| 近似正态 + 线性关系 | **Pearson** |
| 定距、定比类型 | **Pearson** |
| 不正态 或 非线性但单调 | **Spearman** |
| 变量是定类数据（如男女） | **Spearman** |
| 存在极端值、异常值 | **Spearman** |

**Step 3：显著性检验**
- 将相关系数进行 t 检验以确定统计显著性
- 在 SPSS 结果中 **p < 0.05** 即通过检验

### Pearson 相关系数适用条件

1. 定距、定比变量
2. 正态分布
3. 线性关系

### Spearman 适用条件

1. 定序与定序变量

### SPSS 操作指南

| 分析方法 | SPSS 操作路径 |
| :--- | :--- |
| **卡方检验** | 分析 → 描述统计 → 交叉表 → 统计 → 卡方（行列各放一个变量，行放因变量） |
| **Eta 系数** | 分析 → 统计 → 相关性 |
| **Pearson 相关** | 分析 → 相关性 → 双变量 → 更改选项 |
| **偏相关分析** | 分析 → 相关 → 偏相关（限制其他变量以得到净相关关系） |
